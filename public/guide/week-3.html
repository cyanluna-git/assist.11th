공유 가능한 링크 생성

0-10분

Next.js 프로젝트 만들기

npx create-next-app@latest trip-dashboard
cd trip-dashboard
npm run dev

브라우저에서 기본 화면 열기

10-20분

프로젝트 구조 보기

  • app/page.tsx 또는 app/page.js 확인
  • 스타일 파일 위치 확인
  • 기본 페이지를 대시보드용으로 바꿀 준비

20-35분

프롬프트 1 기존 화면 옮기기

기존 출장 / 미팅 의사결정 대시보드 구조를
Next.js app router 구조로 옮겨줘
한 페이지로 유지
입력 영역과 결과 카드 3개를 그대로 살려줘
초보자가 읽기 쉬운 파일 구조로 유지해줘

35-45분

프롬프트 2 화면 다듬기

좋아
이제 Next.js 프로젝트 안에서 아래만 고쳐줘
1. 입력 폼 간격 정리
2. 카드 3개를 더 보기 쉽게
3. 로딩과 오류 문구 더 명확하게

45-52분

GitHub에 올리기

git init
git add .
git commit -m "Build Next.js dashboard"
git remote add origin [내 저장소 주소]
git branch -M main
git push -u origin main

52-60분

Vercel 첫 배포

  • Vercel 가입 또는 로그인 페이지를 새 탭으로 열기
  • 가입 또는 로그인 완료 후 현재 교안 탭으로 돌아오기
  • GitHub 저장소 선택
  • 기본 설정으로 배포 시작
  • 배포 링크 열고 실제 동작 확인
