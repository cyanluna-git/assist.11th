필수 계정

GitHub 가입 필수! Claude Desktop 준비 필수! Vercel은 후반 단계

GitHub는 저장과 제출 Claude Desktop은 1주차 공통 도구 Vercel은 배포 단계에서 사용

공개 가이드

  • 이 가이드는 로그인 없이 바로 열 수 있는 공개 페이지로 운영
  • Claude Desktop 설치와 로그인까지 끝내면 1주차를 바로 시작할 수 있음
  • Vercel 가입 링크는 새 탭에서 열리고, 가입 후 지금 탭으로 돌아오면 됨
  • 메인 가이드와 3주차 배포 교안도 로그인 없이 바로 열람 가능

1. GitHub 가입

먼저 GitHub 계정 만들기

  1. GitHub 공식 가입 페이지 열기
  2. 이메일, 비밀번호, 사용자명, 국가/지역 입력
  3. 이메일 인증 완료
  4. 로그인 후 프로필은 나중에 정리해도 됨

핵심, 가입 완료, 이메일 인증 완료

2. Claude Desktop 준비

1주차 공통 도구 설치와 로그인

  1. Claude Desktop 다운로드 페이지 열기
  2. 운영체제에 맞는 버전 설치
  3. Claude Desktop 실행 후 로그인 완료
  4. 새 대화를 시작할 수 있는 상태 확인

핵심, 1주차는 Claude Desktop 기준으로 함께 진행

3. Vercel 가입

배포 주차 전에 Vercel 계정 만들기

  1. Vercel 공식 가입 페이지를 새 탭으로 열기
  2. 가능하면 GitHub 계정으로 연결해서 가입
  3. 첫 프로젝트 생성 단계는 나중에 진행해도 됨
  4. 가입이 끝나면 이 가이드 탭으로 돌아와서 다음 단계 진행
  5. 배포 단계에서 GitHub 저장소 연결

핵심, 배포 직전에 준비해도 충분함

GitHub 추가 설정

이메일 인증 완료, 2단계 인증까지 하면 더 좋음

GitHub 계정 만들기 이후, 이메일 인증 먼저, 가능하면 2단계 인증까지 설정

Vercel 추가 설정

가능하면 GitHub 연결로 시작

이메일 가입도 가능, 하지만 첫 프로젝트 단계에서 결국 Git provider 연결 필요 수업에서는 GitHub 연동이 가장 자연스러운 흐름

로그인은 Vercel 탭에서 진행 · 설명은 이 가이드 탭에서 계속 확인
