터미널에서 기록 남기기

0-10분

WSL에서 저장소 가져오기

git clone [내 저장소 주소]
cd [프로젝트 폴더]
ls

파일과 폴더가 보이면 시작 가능

10-20분

개발 서버 실행

npm install
npm run dev

브라우저에서 localhost 화면 열기

20-30분

현재 프로젝트 상태 읽기

git status
pwd
node -v
npm -v

지금 어디에서 무엇을 다루는지 확인

30-40분

CLI용 수정 요청 예시

현재 Next.js 대시보드에서 아래만 고쳐줘
1. 입력 폼 라벨 더 명확하게
2. 로딩 상태 더 보기 쉽게
3. 결과 카드 간격 정리
4. 기존 구조는 최대한 유지

40-50분

변경 내용 확인

git status
git diff

무엇이 바뀌었는지 읽고 이해하기

50-60분

커밋과 push

git add .
git commit -m "Refine dashboard in WSL"
git push
