사전 준비

수업 당일에는 설치보다 실행과 수정에 시간을 쓰기 위해 미리 준비한다.

이번 과정에서는 OpenAI API Key 같은 별도 유료 API 키를 강제하지 않는다. 대신 GitHub, Git, Node.js, Python, WSL, Claude Desktop, IDE를 준비해두는 것이 중요하다.

GitHub 계정

프로젝트 저장과 제출의 기준점이다.

Git

git --version 으로 확인한다.

Node.js LTS

node -v, npm -v 로 확인한다.

Python 3

python --version 또는 python3 --version 으로 확인한다.

WSL

Windows 사용자라면 wsl -l -v 로 상태를 확인한다.

Claude Desktop + IDE

1주차는 Claude Desktop, 2주차부터는 VS Code나 Cursor 같은 IDE를 사용한다.
