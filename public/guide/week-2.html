Week 2 Flow

IDE에서 API 연결 후 GitHub까지

1 IDE 열기

Replit 결과물을 로컬로 가져오기

→
2 API 연결

날씨 공휴일 환율 데이터 붙이기

→
3 테스트

로딩 오류 빈 상태 확인

→
4 Push

assist-11th 저장소에 올리기

0-10분

프로젝트 가져오기

  • 1주차 Replit 코드를 로컬 폴더로 저장
  • IDE에서 프로젝트 폴더 열기
  • index.html style.css script.js 위치 확인

10-20분

GitHub 저장소 만들기와 첫 연결

git init
git add .
git commit -m "Start dashboard project"

가능하면 assist-11th 안에서 저장소 만들기

20-30분

프롬프트 1 날씨 API 연결

현재 프로젝트 구조를 유지하면서
도시 입력을 기준으로 Open-Meteo 날씨 데이터를 불러오게 해줘
응답이 없을 때는 안내 문구도 보여줘
초보자가 읽기 쉬운 코드로 유지해줘

30-40분

프롬프트 2 공휴일과 환율 추가

좋아
이제 Nager.Date로 공휴일 카드 추가
Frankfurter로 환율 카드 추가
카드 3개가 같은 형식으로 보이게 맞춰줘

40-50분

프롬프트 3 로딩 오류 빈 상태 정리

입력값이 없을 때
도시 또는 국가 정보가 맞지 않을 때
API 응답이 늦을 때
각 상황에 맞는 문구와 화면 상태를 정리해줘

50-60분

GitHub에 push

git remote add origin [내 저장소 주소]
git branch -M main
git push -u origin main

저장소 링크 열어서 파일이 올라갔는지 확인
