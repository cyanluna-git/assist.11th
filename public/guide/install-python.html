Python 설치 가이드

Python 설치 필수. 자동화, 스크립트, 확장 실습용 기본 도구. Windows 기준은 python.org 공식 설치

설치 순서

  1. Python 공식 다운로드 페이지 열기
  2. 최신 안정 버전 Download 버튼 클릭
  3. 설치 파일 실행
  4. 간단 설치 또는 Customize installation 선택
  5. 설치 후 새 터미널 열기

PATH 안내

PATH 추가, 초보자라면 켜두기

공식 문서 기준으로 optional. 하지만 python 명령을 바로 쓰려면 켜두는 편이 덜 헷갈림.

설치할 때 보기

체크박스가 보이면 PATH 추가 켜두기

수업에서는 python, python3, py 명령 중 하나를 바로 실행. PATH 옵션 켜두면 이해가 쉬움.

설치 후 확인

세 가지 명령 중 하나 이상 동작 확인

python --version
python3 --version
py --version

Windows에서는 py 가 먼저 동작하는 경우도 있음.

설치 확인

python --version

또는

python3 --version

추가 확인

py --version

Windows에서는 Python launcher와 함께 py 명령 사용 가능.

공식 문서 메모

  • PATH 추가, 공식 문서 기준 optional
  • 모든 사용자 설치가 필요하면 Customize installation 사용
  • 기본 개인 설치는 관리자 권한 없이 가능

자주 막히는 경우

  • python 이 안 되면 py 또는 python3 로 확인
  • 설치 직후 명령이 안 되면 새 터미널 열기 또는 재부팅
