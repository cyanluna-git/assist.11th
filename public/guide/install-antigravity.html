Antigravity 설치 가이드

Antigravity 설치 가이드. 개인 Gmail 계정 준비. Mac, Windows, 일부 Linux 배포판에서 로컬 설치 필요

설치 전에 준비

  • 개인 Gmail 계정
  • Chrome 브라우저
  • 로컬 설치 가능한 개인 PC

공식 안내

Google 공식 codelab 기준, 로컬 설치형, 다운로드 페이지에서 운영체제에 맞는 버전 선택

설치 순서

  1. Antigravity 다운로드 페이지 열기
  2. 운영체제에 맞는 버전 선택
  3. 설치 파일 실행, 설치 완료
  4. 초기 설정에서 Next 따라가기
  5. 필요하면 VS Code 또는 Cursor 설정 가져오기, 아니면 새로 시작

첫 실행 후

  1. 테마 선택
  2. 에이전트 사용 방식 확인
  3. Browser 기능용 Chrome 확장 설치
  4. Playground 또는 Workspace 열기
