스킬이란 실습으로

이번 강의 목표

스킬로 작업 기록 자동화

  • 스킬 개념 이해
  • worklog.md 직접 제작
  • /worklog 호출해보기
  • worklogs/ 폴더 git 커밋

사전 확인

  • Claude Code 설치 완료
  • git 강의 완료
  • ~/.claude/ 폴더 존재 여부

확인 → ls ~/.claude/skills/

스킬이란

반복 프롬프트를 파일로 저장한 것

없을 때 매번 설명

긴 프롬프트 매번 입력
조금씩 달라짐
실수 가능성 높음

→
스킬 있을 때 /worklog

한 줄로 호출
항상 동일하게 동작
어느 프로젝트서나 사용

핵심

~/.claude/skills/ 폴더의 .md 파일 하나 = 나만의 명령어

실습 흐름

Claude Code에게 지시하면서 하기

스킬 파일을 직접 만들고 → 호출해보는 순서

0–5분 · Step 1

스킬 폴더 확인

Claude Code에 입력

ls ~/.claude/skills/
  • 폴더가 없으면 Claude Code가 자동 생성
  • 이미 있는 스킬 파일들 확인

5–20분 · Step 2

worklog.md 스킬 파일 만들기

Claude Code에 입력

~/.claude/skills/worklog.md 파일을 만들어줘.
이 스킬의 역할: /worklog 라고 입력하면
오늘 날짜(YYYY-MM-DD)로 worklogs/ 폴더에 마크다운 파일을 만들고,
오늘 작업한 내용과 다음에 할 일을 정리해서 저장해줘.
파일이 이미 있으면 새 항목을 추가해줘.
  • ~/.claude/skills/worklog.md 파일 생성 확인
  • 내용이 의도대로인지 검토

생성된 스킬 파일 예시

# /worklog — 작업 로그 저장 스킬

오늘 날짜로 `worklogs/YYYY-MM-DD.md` 파일을 만들어줘.

## 파일이 없으면
새로 생성. 아래 템플릿으로:

# YYYY-MM-DD 작업 로그

## 오늘 한 것
-

## 다음에 할 것
-

## 메모


## 파일이 이미 있으면
파일 맨 아래에 `---` 구분선 추가 후 새 항목 추가.

완료 후: "worklogs/YYYY-MM-DD.md 저장 완료" 출력.

핵심 체험 · 내 첫 번째 스킬 파일 완성

20–30분 · Step 3

/worklog 실행

Claude Code에 입력

/worklog
  • worklogs/2026-04-25.md 파일이 생겼는지 확인
  • 오늘 날짜로 된 마크다운 파일인지 확인
  • 파일 내용 열어서 템플릿 구조 확인

핵심 체험 · 내가 만든 명령어가 동작하는 감각

30–40분 · Step 4

기록이 쌓이는 구조

작업할 때마다 /worklog → 날짜별 기록 자동 누적

내 프로젝트/
├── index.html
├── worklogs/
│   ├── 2026-04-25.md
│   ├── 2026-04-26.md
│   └── 2026-04-27.md
└── ...

Claude Code에 입력 (다음 세션 시작 시)

worklogs/ 폴더의 가장 최근 파일을 읽고 어디까지 했는지 파악한 뒤 이어서 진행해줘.

40–50분 · Step 5

코드 + 기록 같이 커밋

Claude Code에 입력

오늘 작업 내용 /worklog로 기록하고
코드 변경사항이랑 같이 커밋해줘.
  • 코드 변경 + 작업 로그가 같은 커밋에 묶임
  • GitHub에서 작업 흐름 한눈에 파악 가능

핵심 · 코드 기록(git) + 작업 흐름 기록(worklog) = 완전한 이력

50–60분 · 마무리

오늘의 습관 3가지

① 작업 시작 전

worklogs/ 최근 파일 읽히기
컨텍스트 복원

→
② 작업 중

/worklog 수시로
진행상황 기록

→
③ 작업 끝나면

코드 + 워크로그
같이 커밋

오늘 한 것

스킬 제작 + 워크로그 습관

  • 스킬 개념 이해
  • worklog.md 스킬 직접 제작
  • /worklog 호출 체험
  • worklogs/ 폴더 구조 파악
  • git 커밋까지

과제 선택

이번 주 안으로

  • 프로젝트에서 /worklog 3회 사용해보기
  • 스킬 파일 내용 수정해서 나만의 버전 만들기
  • ~/.claude/skills/ 에 다른 스킬도 추가해보기

핵심 · 스킬은 만들고 쓸수록 좋아진다
