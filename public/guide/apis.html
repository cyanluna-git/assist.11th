첫 프로젝트

출장 / 미팅 의사결정 대시보드

MBA · 스타트업 수강생에게는 단순 TODO 앱보다, 의사결정에 닿는 데이터 앱이 더 적합하다. 첫 프로젝트는 도시, 날짜, 국가를 입력하면 날씨, 공휴일, 환율을 같이 보는 작은 대시보드로 잡는다.

입력

  • 도시
  • 국가
  • 출장 또는 미팅 날짜

출력

  • 날씨와 강수 확률
  • 공휴일 여부
  • 환율
  • 한 줄 요약

Weather

Open-Meteo

날씨, 기온, 강수확률 등 첫 프로젝트에 필요한 데이터가 충분하다.

공식 문서

Holiday

Nager.Date

국가별 공휴일 확인에 적합하다. 별도 인증 절차 없이 바로 시작하기 좋다.

공식 문서

Exchange

Frankfurter

환율 확인용으로 가볍고 명확하다. 입문 실습에 필요한 범위가 충분하다.

공식 문서

왜 이 프로젝트인가

키 발급, 승인 대기, 복잡한 인증 없이도 실제적인 데이터 앱 경험을 줄 수 있다.

첫 단계에서 좋은 이유

  • 실무 맥락이 있다
  • 입력과 출력 구조가 단순하다
  • UI를 빠르게 만들 수 있다
  • 나중에 예산 계산, 도시 비교, 출장 추천으로 확장 가능하다

첫 단계에서 피하는 것

  • 키 발급이 필요한 API
  • 권한 승인 대기가 필요한 공공데이터
  • 지역별 형식이 제각각인 데이터
