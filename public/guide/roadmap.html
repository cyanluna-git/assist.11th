학습 로드맵

브라우저에서 시작해 WSL + CLI까지 이동하는 4-5주 과정.

이 과정은 한 번에 모든 것을 설치하고 설명하는 수업이 아니다. 작은 성공 경험을 만든 뒤, 점진적으로 더 실제적인 개발 환경으로 이동하는 구조다.

1주차

비설치형 첫 경험

  • 바이브코딩의 개념 설명
  • 질문형 사용과 작업지시형 사용 비교
  • Replit 같은 브라우저 도구로 첫 앱 생성
  • 출장 / 미팅 의사결정 대시보드 시작
2주차

GitHub와 프로젝트 개념

  • Git과 GitHub 차이 설명
  • 저장소, 파일, 버전 개념 소개
  • README와 프로젝트 구조 정리
  • GitHub 링크 제출 흐름 익히기
3주차

IDE에서 프로젝트 수정하기

  • VS Code + GitHub Copilot 기본 시연
  • 학생은 VS Code, Cursor, Antigravity 등 자유 사용
  • 여러 파일 수정 경험
  • 기능 1개 추가 과제
4주차

WSL + CLI 입문

  • WSL에서 프로젝트 열기
  • git clone, npm install, npm run dev
  • Claude Code 또는 Copilot CLI 흐름 보기
  • GitHub 저장소 링크 제출
5주차

배포와 확장

  • Vercel 배포 소개
  • 브라우저, IDE, CLI, 배포 환경 비교
  • 다음 단계 로드맵 제시
