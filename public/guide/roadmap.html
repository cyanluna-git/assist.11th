학습 로드맵

Claude Desktop에서 시작해 WSL + Claude Code까지 이동하는 4-5주 과정.

이 과정은 한 번에 모든 것을 설치하고 설명하는 수업이 아니다. 작은 성공 경험을 만든 뒤, 점진적으로 더 실제적인 개발 환경으로 이동하는 구조다.

1주차

Claude Desktop 첫 경험

  • 바이브코딩의 개념 설명
  • 질문형 사용과 작업지시형 사용 비교
  • Claude Desktop으로 첫 앱 초안 생성
  • 출장 / 미팅 의사결정 대시보드 시작
2주차

IDE와 API 연결

  • Claude Desktop 산출물을 로컬 파일로 옮기기
  • IDE에서 프로젝트 구조 읽기
  • 공개 API 연결
  • GitHub에 push까지 완료
3주차

Next.js와 Vercel 배포

  • 정적 페이지에서 프레임워크로 이동
  • Next.js 기본 구조 이해
  • GitHub 저장소를 Vercel에 연결
  • 첫 배포 링크 만들기
4주차

WSL + CLI 입문

  • WSL에서 프로젝트 열기
  • git clone, npm install, npm run dev
  • Claude Code 흐름 보기
  • 터미널에서 직접 수정 흐름 체험
5주차

Backend와 DB 맛보기

  • 저장 기능이 왜 필요한지 설명
  • 백엔드와 데이터베이스 역할 구분
  • 다음 단계 로드맵 제시
